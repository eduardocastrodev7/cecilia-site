# Cecília Digital

Site institucional da Cecília Digital — Engenharia e Infraestrutura para E-commerce.

## Tecnologias

- **Next.js** (App Router)
- React
- TypeScript
- Tailwind CSS
- shadcn/ui
- Framer Motion

## Desenvolvimento

```sh
pnpm install
pnpm dev
```

O servidor de desenvolvimento sobe em [http://localhost:8080](http://localhost:8080).

## Scripts

| Comando       | Descrição                    |
|---------------|------------------------------|
| `pnpm dev`    | Inicia o servidor de desenvolvimento (porta 8080) |
| `pnpm build`  | Gera o build de produção     |
| `pnpm start`  | Inicia o servidor em modo produção |
| `pnpm lint`   | Executa o ESLint             |
